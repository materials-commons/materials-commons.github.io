mc mkdir

Make directories

usage: mc mkdir [-h] [-p] [--remote-only] [paths ...]

Positional Arguments

paths

Directory names

Default: ['/mnt/home2/gtarcea/workspace/src/github.com/materials-commons/mccli']

Named Arguments

-p

Create intermediate directories as necessary

Default: False

--remote-only

Make remote directories only. Does not compare to local tree.

Default: False
