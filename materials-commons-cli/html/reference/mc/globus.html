mc globus¶

mc globus¶

Manage Globus transfers

usage: mc globus <transfertype> [<args>]

The transfer types are:
  download   Manage Globus downloads                 
  upload     Manage Globus uploads

Positional Arguments¶

transfertype

Type of transfer to manage.

Named Arguments¶

--set-globus-endpoint-id

Set local globus endpoint ID

--clear-globus-endpoint-id

Clear local globus endpoint ID

Default: False

mc globus upload¶

List, create, finish, and delete Globus uploads. By default lists current project uploads. With –all lists all Globus uploads.

usage: mc globus upload [-h] [--id] [--owner] [-d] [--regxsearch] [--sort-by [SORT_BY [SORT_BY ...]]] [--json] [-o OUTPUT] [-f] [--remote EMAIL URL] [--all]
                        [--create] [--delete] [--name NAME] [--finish] [--goto] [--set] [--unset]
                        [expr [expr ...]]

Positional Arguments¶

expr

select Uploads that match the given regex (default matches name)

Named Arguments¶

--id

match id instead of name

Default: False

--owner

match owner instead of name

Default: False

-d, --details

print detailed information

Default: False

--regxsearch

use regular expression search instead of match

Default: False

--sort-by

columns to sort by

Default: [‘name’]

--json

print JSON data

Default: False

-o, --output

output to file

-f, --force

force overwrite of existing output file

Default: False

--remote

Remote to upload files to

--all

list Uploads from all projects

Default: False

--create

create a Upload

Default: False

--delete

delete a Upload, specified by id

Default: False

--name

New upload name, for use with –create. If not given, a random name will be generated.

Default: “”

--finish

Finish an upload. This closes the upload to new files and adds uploaded files to the project.

Default: False

--goto

Open the Globus File Manager in a web browser.

Default: False

--set

Set the Globus upload used by mc up –globus.

Default: False

--unset

Unset the Globus upload used by mc up –globus.

Default: False

usage: mc globus download [-h] [--id] [--owner] [-d] [--regxsearch] [--sort-by [SORT_BY [SORT_BY ...]]] [--json] [-o OUTPUT] [-f] [--remote EMAIL URL]
